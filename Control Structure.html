1.Conditional Statements:

Conditional statements are used in performing different actions based on different conditions.

if, else, and else if

1.If statement:

is used when you want to execute a block of code for a true condition.

2.Else statement:

is used when you want to execute a block of code for a false condition.

3.Else if statement:

is used when you want to execute a block of code to take off when the first condition is false.

4.Switch statement:

is used to declare many alternative blocks of code to be executed.
